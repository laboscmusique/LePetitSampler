# LePetitSampler

A browser-based sampler built with vanilla JS and Web Audio API.

## Features

- **Chromatic Edition** — load a sample tuned in C3 and play it at different pitches
- **Drum Edition** — 8 pads with individual samples, pitch, volume, ADSR, and a step sequencer (32 steps, 4 measures export)
- Real-time ADSR envelope with visual graph
- Waveform editor (sample start, loop in/out)
- Filter module (LP/HP)
- Computer keyboard support (QWERTY/AZERTZ/QWERTZ)
- Performance recording (WAV export)
- 6 languages (FR, EN, DE, ES, IT, PT)

## Dev

```bash
npm install
npm run dev      # build + watch + serve
npm run build    # build to app.js
```
